The Toolbar

The toolbar is something that you will be familiar with if you have used any Adobe programs. If you aren't familiar with the toolbar, that's alright! There are only a couple tools that you should be familiar with.

The toolbar is usually located on the left side of your screen. The 4 main tools that I use while painting are:

  1. the brush tool
  2. the color picker
  3. the freehand selection tool
  4. the transform tool
The brush tool is used to put down and erase colors, the color picker, well, picks colors, the freehand selection tool allows you to draw and a certain area on the current layer, and the transform tool allows you to move and resize that selection.

The Color Picker

The color picker is an essential part of digital painting. If you have never painted digitally before, this is where you will be selecting colors to paint with.

On the top right section of your screen, you will see a color picker, which is what you use to select colors. By clicking on the small icon to the top left of the color picker, you can change the shape of it. I personally like to use the second one, which has the hue on the outside and the saturation and lightness of that hue on the inside.

One very useful shortcut is if you have the brush tool selected, holding down the Ctrl key will bring up the color picker. You can click anywhere on your canvas and it will select the color that is on the canvas, and releasing Ctrl will go back to the brush tool where you can paint with your newly selected color.
